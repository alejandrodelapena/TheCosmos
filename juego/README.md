# Juego — El mundo de Caine

Aquí irá el videojuego (el "circo digital") hecho con **Unity** y **C#**.
Lo empezarás en el PC de sobremesa, cuando ya manejes lo básico.

## Cuando llegue el momento

1. Instala **Unity Hub** + una versión LTS de Unity.
2. Crea el proyecto Unity **dentro de esta carpeta** (`juego/`).
3. La IA (carpeta `ia/`) y el juego se comunicarán por mensajes JSON;
   el formato está descrito en `../compartido/eventos.md`.

> Nota: los archivos pesados que genera Unity (Library/, Temp/, Build/...)
> ya están ignorados en el `.gitignore`, así que no ensuciarán el repo.
